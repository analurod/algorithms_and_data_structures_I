# 🚀 DSA I - 2024.1

## 📌 Overview
This repository gathers the exercises and materials developed throughout the **Algorithms and Data Structures I (DSA I)** course in the Computer Science program. The focus is on implementing classical algorithms and the practical application of fundamental data structures such as queues, stacks, lists, trees, and graphs.

---

## 📚 Course Content

- **Introduction to Algorithms**
- **Basic Data Structures**
  - Arrays
  - Linked Lists
  - Stacks
  - Queues
- **Search and Sorting Algorithms**
- **Recursion**
- **Trees and Graphs**

---

## 🔑 Features
- 📌 Commented solutions for URI/Beecrowd exercises.
- 📌 Organization by problem statement and exercise number.
- 📌 Practical approach to essential course concepts.

---

## 🛠️ Technologies Used
